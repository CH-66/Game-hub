# 里程碑 1：规则引擎 + 本地对战

## 目标

完成跳跳棋核心规则、可在本地进行 2 人轮流对战的基础 UI。

- [x] 实现回合与状态流转
- [x] 实现胜负判定
- [x] 渲染棋盘 UI 与落子交互
- [x] 添加本地对战基本提示（回合、提示信息）

## 交付物

- 棋盘数据结构与渲染
- 棋子移动规则：单步、跳跃、多跳
- 胜负判定
- 本地回合切换与提示

## 验收标准

- 本地两名玩家可完成一局对战
- 走子合法性正确（多跳可用）
- 胜负判定准确

## 关键模块

- `rules/board.ts`：棋盘与初始布局
- `rules/move.ts`：合法走子与多跳逻辑
- `rules/win.ts`：胜负判定
- `ui/Board.tsx`：棋盘渲染与交互
